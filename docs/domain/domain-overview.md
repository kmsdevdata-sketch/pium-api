## Layer'd 도메인 
- 레이어드는 사용자가 피부 진단을 통해 상품,루틴을 추천 받는 서비스이다 
- 레이어드는 사용자의 피부상태에 맞는(최적화된) 상품과 루틴을 
추천 하여 추천된 상품/루틴 을 통해 사용자 피부상태의 실질적 개선을 가져오는것을 목표로 한다
- 레이어드의 상품/루틴 추천을 받기 위해서는 설문을 해야한다 
  - 회원이 아닌(비로그인) 상태에도 설문이 가능하다 
  - 회원이 아닌 상태에도 preview(진단 결과 미리보기)확인이 가능하다
  - 설문을 한뒤에 로그인을 하면 프리뷰가 아닌 전체 진단결과 확인이 가능하다 
- 회원에게 상품/루틴을 추천하는 기준점은 회원의 피부상태에 필요한 성분군을 사용한다
  - 회원의 피부상태를 산출하는 방법은 설문(Survey)를 통하여 산출한다
  - 설문을 통하여 총 6가지 기준의 상태점수를 산출한다 
    - 건조 / 피지 / 트러블 / 민감 / 색소 / 노화 
    - DRYNESS / SEBUM / ACNE / SENSITIVITY / PIGMENTATION / AGING
  - 상태 점수를 기반으로 사용자에게 필요한 성분군 총 8가지의 점수를 산출한다 
    - 여드름 케어 / 피지조절 / 진정 / 수분 공급 / 피부 장벽 강화 / 미백&톤 / 각질 제거 & 재생 / 주름개선 & 탄력강화 
    - ACNE / SEBUM_CONTROL / SOOTHING / HYDRATION / BARRIER / BRIGHTENING / TURNOVER / ANTI_AGING
- 상품을 추천하는 기준점은 상품의 성분군 및 상품특징을 반영한 상품의 성분군 점수를 활용한다 
  - 피부제품은 상세 성분군 함량을 제공하지 않는 도메인 특수성을 고려하여 기재된 상품의 성분모음과
해당 성분들의 순서를 고려하여(앞순서일수록 함량이 많이 되어있음) 산출한다 - gpt이용 
  - 상품의 성분군 기준점은 사용자 필요 성분군 목록과 동일하다 
  - 상품 성분군과 사용자 필요 성분군을 매칭하여 벡터유사도를 기반으로 계산한다 
    - 벡터유사도 기반 계산을 하지만 상품 수량이 적을시 노출되는 상품의 수량을 고려하여 유기적 변경 가능하다
- 루틴에 제공되는 목록군은 사용자 피드백을 기반으로 개선 예정
- 루틴추천에서 각 목록군은 해당 목록에서 사용자 추천 상품1위와 매칭하여 제공한다
