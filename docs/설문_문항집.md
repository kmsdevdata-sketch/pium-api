## 전체 설문 문항 (최종안)

---

### 섹션 A. 피부 상태

---

**Q1. DRYNESS ① — 건조 빈도**

> 세안 후 아무것도 바르지 않으면, 피부가 얼마나 당기나요?

```
① 거의 안 당겨요
② 약간 당기는 느낌이에요
③ 꽤 당기고 불편해요
④ 너무 당겨서 표정 짓기도 힘들어요
```

**측정 의도**
"세안 후 보습 없이"라는 조건을 명시한 이유는 보습 루틴이 건조 인식을 가려서 
실제 피부 상태를 왜곡하는 경우가 많기 때문이다. 이 문항은 TEWL(경표피 수분 손실) 경향을 간접 측정하는 개념

---

**Q2. DRYNESS ② — 건조 패턴**

> 피부가 가장 건조하게 느껴지는 때는 언제예요?

```
① 세안 직후
② 외출하고 돌아왔을 때
③ 자기 직전에도 건조해요
④ 하루 종일 비슷하게 건조해요
⑤ 건조한 느낌이 거의 없어요
```

**측정 의도**
건조발생 원인을 파악하기 위해.
- 세안 직후 → 세정 과정에서 피부 지질층 손상 가능성 (BARRIER 파생에 기여)
- 외출 후 → 환경적 수분 증발 취약 (BARRIER + DRYNESS)
- 자기 직전까지 → 내인성 건조, 보습막 형성 능력 자체 부족 (DRYNESS 강도 높음)
- ⑤ 선택 시 → DRYNESS 점수 하향 보정

---

**Q3. OILINESS ① — 유분 분포**

> 오후쯤 됐을 때, 피부 상태가 어떤 편이에요?

```
① 여전히 건조하거나 당겨요
② 딱히 이상 없어요
③ T존(이마·코)만 번들거려요
④ 볼까지 전체적으로 번들거려요
```

**측정 의도**
- T존과 U존(볼·턱) 유분 분포 패턴은 피지선 분포와 직결.
- T존만 번들거리는 패턴은 복합성 피부의 전형적 신호
- 전체 번들거림은 피지 과분비 경향으로 해석
- ①을 선택하면 DRYNESS 보정 신호로 활용.

---

**Q4. OILINESS ② — 유분 촉발 요인**

> 유분이 특히 많아진다고 느낄 때가 있나요? (복수 선택)

```
□ 덥거나 습한 날
□ 스트레스받을 때
□ 특별한 이유 없이 항상 번들거려요
□ 그런 느낌이 없어요
```

**측정 의도**
복수 선택을 허용한 이유는 피지 분비는 단일 원인이 아니라 열, 안드로겐, 스트레스 호르몬 등 복합 요인으로 발생하기 때문
"항상 번들"은 OILINESS 고점 처리, "느낀 적 없다" 단독 선택은 OILINESS 저점 처리

---

**Q5. BLEMISH ① — 트러블 빈도**

> 여드름, 뾰루지 같은 트러블이 얼마나 자주 생겨요?

```
① 거의 안 생겨요
② 한 달에 1~2개 정도
③ 매주 새로 생기는 편이에요
④ 항상 여러 개 있어요
```

**측정 의도**
빈도를 "개수 + 주기" 조합으로 표현한 이유는 단순 개수보다 발생 주기가 만성 트러블 경향을 더 잘 반영하기 때문
④는 BLEMISH_PRONENESS 고점이면서 동시에 SENSITIVITY gating 여부를 함께 검토해야 하는 신호.

---

**Q6. BLEMISH ② — 트러블 부위**

> 트러블이 주로 어디에 생기나요? (복수 선택)

```
□ 이마
□ 코 주변
□ 볼
□ 턱·입 주변
□ 부위 상관없이 전체적으로
□ 거의 안 생겨요
```

**측정 의도**
부위별 트러블 패턴은 원인 추정에 중요한 단서.
- 이마·코 → 피지 과분비 연관 (OILINESS 교차 신호)
- 턱·입 주변 → 호르몬성 트러블 경향
- 볼 → 마찰·건조 유발 트러블 가능성 (BARRIER 교차 신호)
- "전체" 또는 3개 이상 선택 → BLEMISH_PRONENESS 상향 보정

이 문항은 점수 산정보다 **추후 추천 설명 문구 생성**에도 활용 가능.

---

**Q7. SENSITIVITY ① — 제품 반응**

> 새 화장품을 쓸 때, 따가움·붉어짐·가려움 같은 반응이 생기는 편인가요?

```
① 거의 없어요
② 가끔 있어요 (제품에 따라 달라요)
③ 자주 있어요 (새 제품엔 늘 조심해요)
④ 거의 항상 반응이 생겨서 제품 고르기가 힘들어요
```

**측정 의도**
"새로운 제품"이라는 조건을 명시한 이유는 기존에 익숙한 제품 반응과 구별하기 위함.
이 문항은 접촉성 자극 반응 경향을 측정하며, ③④ 응답은 SENSITIVITY 고점 처리와 동시에 **추천 gating의 직접 트리거**가 된다.

---

**Q8. SENSITIVITY ② — 환경 자극 반응**

> 건조한 날씨, 바람, 추위 같은 환경 변화에 피부가 민감하게 반응하나요?

```
① 전혀 그렇지 않아요
② 약간 그런 편이에요
③ 꽤 그런 편이에요
④ 계절 바뀔 때마다 피부 상태가 크게 달라져요
```

**측정 의도**
외부 환경 자극 반응은 피부 장벽 기능과 직결.
Q7이 화학적 자극(성분) 민감도를 본다면, Q8은 물리적·환경적 자극 민감도를 본다.
두 문항을 합산해 SENSITIVITY 점수를 구성하고, Q8 고점은 **BARRIER 파생 점수에도 기여**.

---

### 섹션 B. 피부 고민

---

**Q9. PIGMENTATION_TONE — 색소·톤**

> 잡티, 색소침착, 피부 톤 불균일이 신경 쓰이나요?

```
① 전혀 신경 안 써요
② 약간 신경 쓰여요
③ 꽤 신경 쓰여요
④ 지금 제일 큰 고민이에요
```

**측정 의도**
색소침착은 자가 진단이 비교적 명확한 영역이라 단일 문항으로도 충분한 신호를 얻을 수 있음.
④ 선택 시 GOAL 문항의 미백·톤업 가중치와 교차 확인.

---

**Q10. AGING_SIGNS — 노화 징후**

> 잔주름, 탄력 저하, 피부 처짐이 느껴지나요?

```
① 전혀 안 느껴져요
② 약간 느껴져요
③ 꽤 느껴지고 신경 쓰여요
④ 지금 제일 큰 고민이에요
```

**측정 의도**
노화 징후도 자가 인식이 명확한 영역.
다만 연령과의 상관관계가 높아서, 나중에 연령 데이터를 수집할 수 있게 되면 이 문항의 가중치를 연령 보정할 여지를 남겨두는 베스트.

---

### 섹션 C. 목표

---

**Q11. GOAL — 개선 목표**

> 지금 피부에서 가장 바꾸고 싶은 게 어떤건가요? (최대 2개)

```
□ 보습·수분감
□ 트러블·여드름
□ 피부 톤·미백
□ 모공·피지
□ 탄력·주름
□ 민감한 피부 진정·장벽 강화
```

**측정 의도**
GOAL은 SkinMetric 점수를 override하지 않는다. 대신 **추천 가중치 조정**에 사용.
예를 들어 DRYNESS 점수가 낮더라도 GOAL에서 "보습"을 선택했다면 보습 제품의 추천 우선순위를 소폭 올리는 방식.
최대 2개 제한을 둔 이유는 전부 선택하는 응답을 방지해 실제 우선순위 신호를 얻기 위해서.

## 문항 → SkinMetric 매핑 요약

```
DRYNESS           ← Q1 (주), Q2 (보정), Q3①(교차 보정)
OILINESS          ← Q3 (주), Q4 (보정)
BLEMISH_PRONENESS ← Q5 (주), Q6 (보정)
SENSITIVITY       ← Q7 (주), Q8 (주)
BARRIER           ← Q7 + Q8 파생, Q2 세안직후 선택 시 가중
PIGMENTATION_TONE ← Q9 (직접)
AGING_SIGNS       ← Q10 (직접)
GOAL vector       ← Q11 (추천 가중치)
```
